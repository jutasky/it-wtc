### 查看使用
```bash
git clone git@github.com:zp29/wtc-it.git
// 克隆到本地
cd wtc-it
// 进入目录
npm i gulp -g && npm i
// 安装依赖，国内使用 cnpm 
gulp
```

打开浏览器 [http://localhost:8080/](http://localhost:8080/)
